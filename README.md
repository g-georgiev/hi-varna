# Hi Varna! - Student Support Services

Your First Friend in Varna, Bulgaria.

## About

Hi Varna! is a student support service helping international students navigate their educational journey in beautiful Varna, Bulgaria. From university applications to settling in your new home.

## Website

This repository contains the official Hi Varna! website - a single-page responsive site showcasing our services and contact information.

## Features

- Responsive design (mobile, tablet, desktop)
- Service catalog
- Pricing information
- Contact form (powered by Formspree)
- Smooth animations

## Technologies

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript
- Formspree for form handling

## Setup

This is a static website - simply open `index.html` in a browser or deploy to any static hosting service.

### Formspree Configuration

To enable the contact form:
1. Sign up at https://formspree.io
2. Create a new form
3. Replace `YOUR_FORM_ID` in `index.html` with your actual Formspree form ID
